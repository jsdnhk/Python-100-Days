瀏覽器中的JavaScript：

  • ECMAScript: JavaScript語法規範
  • BOM: 瀏覽器對象模型（Browser Object Model）,把瀏覽器當成一個對象（window），通過這個對象可以操控瀏覽器
  • DOM: 文檔對象模型（Document Object Model），把整個頁面當成一個對象（document），通過這個對象可以操作整個頁面
